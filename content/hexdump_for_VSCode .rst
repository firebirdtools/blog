修改字节

安装
*************************************

1. 安装Visual Studio代码（1.17.0或更高版本）
2. 启动代码
3. 从命令面板ctrl+shift+p（Windows，Linux）或cmd+shift+p（OS X）
4. 选择 Install Extension
5. 选择扩展名 hexdump for VSCode
6. 重新加载Visual Studio代码
